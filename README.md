slim-quickstart
===============

Slim Framework Quickstart


## Installation

Slim-Quickstart is available via Composer:

```cli
$ cd /path/to/your/project
$ composer create-project --no-dev dmitrymomot/slim-quickstart ./ ~1.0
```
